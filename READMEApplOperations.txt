1. To start the application, run the GUI.java file present in the java/ folder.

2. Once the program starts running, you are prompted to identify yourself as either a Dealer, Customer, or Administrator. Clicking on any of these three options will progress the application.

3. If you select yourself as a Dealer, you can now access various functions regarding your position, such as ordering from a manufacturer, vehicle locator services, and sales history

4. If you select yourself as a Customer, you are brought to an "ordering a car" screen, where you go through the process of finalizing a purchase for a new vehicle.

5. Lastly, if you select yourself as an Administrator, you are brought to a screen with a text box where you can enter SQL statements and fetch the resulting tables.
